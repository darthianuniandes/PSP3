Ejecute el programa desde el directorio raíz(aquí se encuentran la carpeta src y target) utilizando el comando: 

java -cp target/PSP4Ranges-1.0-SNAPSHOT.jar co.com.ism.controller.RangesApp proyectoPrueba/ proyectoPruebaLibro/


La aplicación soporta que ingrese la ruta completa de cualquier proyecto. Para esta prueba se usan por defecto los datos dados en el script de la tarea, es decir los datos de conteo de Loc y paginas del libro.

Nota: en el directorio raíz se encuentran los directorios proyectoPrueba y proyectoPruebaLibro usados para capturar la información de prueba mostrada en el script de la tarea
